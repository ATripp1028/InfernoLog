# InfernoLog

A Geometry Dash demon tracking web application.

## Documentation

See the [docs](/docs) directory for architecture,
data model, and design documentation.

## Stack

- React + TypeScript + Vite (frontend)
- AWS Lambda + SST (backend)
- PostgreSQL via Neon
- AWS Cognito (auth)

## Status

Currently in active development. v1 not yet released.
